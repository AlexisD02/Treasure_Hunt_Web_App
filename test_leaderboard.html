Testing Leaderboard page

How it works:

  • Enter a number to determine the size of the leaderboard.
  • If no input from the user is provided, then the test leaderboard will produce 42 players.
  • Check a box to view the testing leaderboard either sorted or not.
  • If both boxes are checked, then by default the Sorting will be set to true
  • Run Test button to show results.
  • Clear button to clear results

Unit Tests

The Unit Tests make use of the corresponding Testing API available at https://codecyprus.org/th/testing.
